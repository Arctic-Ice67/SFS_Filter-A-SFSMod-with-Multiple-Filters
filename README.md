# SFS_Filter-A-SFSMod-with-Multiple-Filters
A SFSMod with multiple filters.Its filters include: color grading (saturation/contrast/exposure), sharpening, fisheye lens distortion, glow, vignetting, color shift, chromatic aberration, depth of field. And it can also save configuration files and select configuration files. If there are any bugs, please contact me.
